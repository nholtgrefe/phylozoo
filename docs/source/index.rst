PhyloZoo
========

PhyloZoo is a Python package for phylogenetic networks analysis. Phylogenetic networks 
represent evolutionary relationships between organisms, providing explicit representation 
of both splitting events (divergence) and merging events (hybridization, admixture, etc.).

.. note::
   PhyloZoo provides core utilities for working with phylogenetic networks, including
   network manipulation, sequence analysis, network inference, and visualization.
