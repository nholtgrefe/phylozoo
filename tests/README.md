# Tests

Test suite for PhyloZoo. Mirrors the source structure.

## Structure

- `core/` — Tests for core modules (networks, quartets, splits, sequences, distance)
- `viz/` — Tests for visualization
- `utils/` — Tests for utilities
- `io/` — Tests for I/O formats
- `fixtures/` — Shared test data
- `conftest.py` — Pytest fixtures and configuration

For how to run and write tests, see the Testing Guide in the development section of the documentation.
