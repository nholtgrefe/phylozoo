# Source

Installable PhyloZoo package. Use `import phylozoo`, not `import src.phylozoo`.

## Structure

- `phylozoo/core/` — Networks, splits, quartets, sequences, distance
- `phylozoo/viz/` — Visualization (optional, requires matplotlib)
- `phylozoo/utils/` — Utilities, I/O, exceptions

For API details, see the documentation.
